# 📱 Time Check Buddy v2.1.1

## 🎯 프로젝트 개요
**완전한 오프라인 지원**과 **데이터 백업/복원** 기능을 갖춘 전문급 PWA 출퇴근 관리 앱입니다.  
기업에서 실제 사용할 수 있는 수준의 완성도 높은 웹 애플리케이션입니다.

## 🆕 v2.1.3 업데이트 (최신)
### 🐛 버그 수정
- **CSV 엑셀 호환성 개선**: UTF-8 BOM 추가로 엑셀에서 한글 깨짐 현상 완전 해결
- **모달 팝업 텍스트 크기 최적화**: 모바일에서 시간 입력 필드가 컨테이너를 넘치지 않도록 폰트 크기 조정
- **12시간 형식 일관성 유지**: 시간 수정 모달에서 24시간 형식으로 바뀌는 문제 해결
- **반응형 모달 디자인**: 모바일 환경에서 더 나은 사용성을 위한 UI 개선

## ✨ 주요 특징

- ⏰ **실시간 시계**: 대형 디지털 시계로 현재 시간을 명확하게 표시
- 🏢 **원클릭 출퇴근**: 출근/퇴근 버튼으로 즉시 시간 기록

- 🔄 **스마트 병합**: 기존 데이터와 가져온 데이터 자동 통합

- 📲 **앱 설치**: 홈 화면에 추가하여 네이티브 앱처럼 사용
- 🎨 **반응형 디자인**: 모든 기기에서 최적의 사용 경험

## 💾 데이터 관리

### **로컬 데이터 구조**
```json
{
  "2025-08-28": {
    "checkin": "09:00",
    "checkout": "18:00",
    "memo": {
      "checkin": "일찍 출근",
      "checkout": "정시 퇴근"
    }
  }
}
```

### **CSV 백업 형식**
```csv
날짜,출근시간,퇴근시간,출근메모,퇴근메모
2025-08-28,09:00,18:00,"일찍 출근","정시 퇴근"
2025-08-29,09:15,,"지각",""
2025-08-30,09:00,17:30,"평상시","조기 퇴근"
```

**파일명 형식**: `TimeCheckBuddy_YYYY-MM-DD.csv`

### **백업/복원 방법**
1. **백업**: 햄버거 메뉴 → 📤 데이터 내보내기
2. **복원**: 햄버거 메뉴 → 📥 데이터 가져오기 → CSV 파일 선택
